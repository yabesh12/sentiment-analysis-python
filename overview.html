Sentiment Analysis Report for Latest News

This project analyzes today's news headlines and categorizes them as positive, negative, or neutral.

  1. Set Up the Flask Application:

    Initialize the Flask project and set up the necessary environment.

  2. Fetch News Data:

    Retrieve news data from a public API.

  3. Process the Data:

    Perform sentiment analysis on each article title using TextBlob, a Python library that offers simple APIs for natural language processing tasks, including sentiment analysis, text classification, and translation.

  4. Design the Frontend:

    Create a user interface to display the sentiment analysis report effectively.
